### 效果预览 😀
![](https://files.catbox.moe/qs8z4z.gif)

### 安装依赖 🔑

```
yarn install
```

### 愉快开始 🎉

```
yarn run electron:serve
```

### 项目打包 📦 (配合 Inno Setup 可实现安装包界面自定义)

```
yarn run electron:build
```

### 环境问题 😮

- electron 版本 **v13.0.0**
- node 版本 **v16.17.0**  
- node-sass 版本 **v6.0.1**
- sass-loader 版本** v10.2.0**
